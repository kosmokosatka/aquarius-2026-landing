## Aquarius 2026 landing

Статический лендинг на основе транскрипции ролика:
`https://www.youtube.com/watch?v=_FxwzyAXuyM`

### Локальный просмотр

Откройте файл `docs/index.html` двойным кликом, либо запустите простой сервер:

```bash
cd "aquarius-2026-landing/docs" && python3 -m http.server 8080
```

Потом откройте в браузере `http://localhost:8080`.

### Публикация (GitHub Pages)

1. Создайте репозиторий и залейте в него папку `aquarius-2026-landing/`.
2. В GitHub откройте Settings → Pages.
3. Source: Deploy from a branch.
4. Branch: `main` и папка `/docs`.
